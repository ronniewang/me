# 02-出售时间之前你要牢记的三条铁律
:nofooter:

在你出售你的时间之前，一定要想清楚

## 1. 成长才是根本关键

成功并不是终点，财富自由也不是终点，只是我们成长路上的里程碑而已，为了能更好的成长，我们就需要做选择，选择将我们的精力放在什么事情上

选择的标准就是

* 能让我完善哪个已有的能力
* 能让我获得什么新的能力

## 2. 重视价值忽略估值

高估值只能是短期的，长期来看，一切估值都要回归价值本源，决定你价值的就是这个社会的需求，让自己变成有用的人

## 3. 耐心是最重要的

复利是世界第八大奇迹，但是复利的价值需要一段长期的时间才能显示出来，闹耐得住寂寞
